# RadioClock 使用方法

## 画面構成

モニターには必ず16:9のアスペクト比のものをご利用ください。
起動すると、1280x720 のフルスクリーンで時計が表示されます。

- **左側**: アナログ時計（秒ドット + 時マーカー）+ デジタル時計
- **右側**: インジケータ 4 個（GPIO またはキーボードで ON/OFF）

## キーボード操作

| キー | 機能 |
|------|------|
| `1` `2` `3` `4` | インジケータ 1〜4 を ON（押している間のみ） |
| `F` | デジタル時計のフォント切替（7セグ風 ⇔ システムフォント） |
| `A` | 画面右下に IP アドレス表示の ON/OFF |
| `R` | setup.txt の設定を手動リロード |

## GPIO 入力

| GPIO ピン (BOARD) | インジケータ |
|-------------------|-------------|
| 11 | インジケータ 1 |
| 12 | インジケータ 2 |
| 13 | インジケータ 3 |
| 15 | インジケータ 4 |

GPIO は LOW でアクティブ（プルアップ済み）。キーボードと GPIO は OR 動作です。

## Web 設定画面

同じネットワーク上のブラウザからアクセスできます。

| URL | 機能 |
|-----|------|
| `http://<IPアドレス>:8000` | 時計表示設定（インジケータ文字・色、時計の色） |
| `http://<IPアドレス>:8080` | WiFi 設定（スキャン・接続・再起動・シャットダウン） |

各ページの下部に相互リンクがあります。

### 時計表示設定（:8000）

- インジケータ 1〜4 の表示テキスト・文字色・背景色を変更
- 時計の文字色・ドット色を変更
- カラーピッカーまたは RGB 値（0〜255）で指定
- 「設定を保存」で反映（約 5 秒以内に自動反映）

### WiFi 設定（:8080）

- 利用可能な WiFi ネットワークのスキャン・接続
- 保存済みネットワークの管理（削除）
- システム再起動 / シャットダウン
